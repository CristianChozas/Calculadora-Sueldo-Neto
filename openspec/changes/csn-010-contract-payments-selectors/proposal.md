# CSN-010 Crear selector de tipo de contrato y numero de pagas

## Objetivo

Permitir que el usuario configure el tipo de contrato y el numero de pagas anuales desde la UI base del formulario.

## Alcance

- Incorporar selector para `indefinido` y `temporal`.
- Incorporar selector para `12` o `14` pagas.
- Reflejar el impacto inmediato en desempleo de Seguridad Social y sueldo mensual bruto.

## Fuera de alcance

- Validacion avanzada del formulario.
- Calculo reactivo completo del neto.
- Persistencia de estado.
