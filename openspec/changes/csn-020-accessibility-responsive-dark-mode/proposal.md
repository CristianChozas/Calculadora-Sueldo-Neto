# CSN-020 Revision de accesibilidad, responsive y dark mode

## Objetivo

Refinar la usabilidad de la calculadora mejorando semantica, foco visible, responsive y consistencia visual en dark mode.

## Alcance

- Reforzar labels y agrupaciones semanticas en el formulario.
- Añadir foco visible consistente en elementos interactivos.
- Ajustar el layout para viewport movil 375px y tablet 768px.
- Revisar que componentes clave sigan leyendo bien en dark mode.

## Fuera de alcance

- Auditoria automatizada completa de accesibilidad.
- Nuevos temas visuales.
- Rediseño de componentes fuera de lo necesario para el ticket.
