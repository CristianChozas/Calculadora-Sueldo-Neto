# CSN-002 Configurar fuentes, paleta de colores y tokens de diseno

## Objetivo

Definir la identidad visual base del proyecto para que los siguientes tickets compartan una paleta consistente, una tipografia profesional y tokens reutilizables.

## Alcance

- Mantener tipografia base con `Geist`.
- Establecer colores semanticos para fondos, superficies, texto, borde y acento.
- Definir tokens reutilizables para radios, sombras y espaciado.
- Aplicar la identidad a la portada inicial sin invadir tickets de layout funcional.

## Fuera de alcance

- Formularios definitivos.
- Panel de resultados.
- Dark mode manual con toggle.
