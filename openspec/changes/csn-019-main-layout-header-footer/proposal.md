# CSN-019 Crear layout principal, header y footer

## Objetivo

Definir la estructura principal de la aplicacion con un layout claro, encabezado propio y pie de pagina.

## Alcance

- Organizar la pantalla en dos columnas en desktop: formulario a la izquierda y resultados a la derecha.
- Mantener stack vertical en movil.
- Añadir header con el titulo de la app.
- Añadir footer con `Cristian Chozas Diaz - 2025`.

## Fuera de alcance

- Rediseño visual profundo de componentes internos.
- Nuevas funcionalidades de navegacion.
- Ajustes de dark mode y accesibilidad del ticket 020.
