# CSN-001 Inicializar proyecto Next.js con TypeScript

## Objetivo

Crear la base tecnica del proyecto con Next.js, TypeScript, Tailwind CSS y ESLint, dejando la estructura inicial lista para desarrollar los tickets posteriores.

## Alcance

- Inicializar aplicacion Next.js en la raiz del repositorio.
- Configurar `app/`, `components/` y `lib/`.
- Dejar scripts base y dependencias instaladas.
- Mantener `documents/roadmap.md` como fuente del backlog.

## Fuera de alcance

- Implementacion de logica fiscal.
- Diseno visual final.
- Formularios, resultados o deploy.
