# CSN-011 Crear selector de comunidad autonoma

## Objetivo

Permitir que el usuario indique su comunidad autonoma dentro del formulario base y dejar visible que la estimacion fiscal actual usa referencia estatal.

## Alcance

- Incorporar dropdown con 17 comunidades autonomas mas Ceuta y Melilla.
- Añadir aviso inicial indicando que el IRPF autonomico puede variar.
- Integrar la seleccion en la UI actual sin alterar todavia el calculo fiscal.

## Fuera de alcance

- Calculo de tramos autonomicos reales.
- Persistencia de formulario.
- Ajustes legales por territorio foral.
